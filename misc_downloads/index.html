Unzipping Large Files on the Command Line:

Reference panels can then be unzipped into where bridgePRS reference panels are stored using the following command: 

tar -zxvf "REF_PANEL.tar.gz" -C "PATH-TO-BRIDGE-PRS-PANELS"
